#  Paradise Nursery – Online Plant Shop

## About the Project

Paradise Nursery is a simple e-commerce web application built with **React**, **Redux Toolkit**, and **React Router**. The application allows users to browse a collection of house plants, add their favorite plants to a shopping cart, update item quantities, and view the total cost before checkout.

This project was developed as the final assignment for learning React and Redux concepts. While building it, I practiced creating reusable components, managing application state with Redux Toolkit, and implementing navigation using React Router.

---

## Features

- Welcome landing page with a brief introduction
- Navigation bar for easy access to different pages
- Display of multiple plant categories
- Product cards with:
  - Plant image
  - Name
  - Description
  - Price
  - Add to Cart button
- Shopping cart with:
  - Quantity controls
  - Remove item option
  - Individual item totals
  - Overall cart total
- Cart item count displayed in the navigation bar
- Continue Shopping and Checkout actions

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/paradise-nursery.git
```

Move into the project directory:

```bash
cd paradise-nursery
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
